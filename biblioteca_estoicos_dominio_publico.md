# Biblioteca Estoica em Domínio Público

Atualizado em: 2026-02-14

## Escopo e limite histórico

Este arquivo reúne uma **busca ampla** por textos em domínio público dos principais nomes citados antes:
- Zenão de Cítio
- Cleantes de Assos
- Crisipo de Solis
- Sêneca
- Epicteto
- Marco Aurélio

Observação crítica:
- Para **Zenão, Cleantes e Crisipo**, quase tudo sobrevive em **fragmentos** (não em obras completas).
- Para **Sêneca, Epicteto e Marco Aurélio**, há obras completas preservadas e várias edições em domínio público.
- Traduções modernas podem ter copyright; priorizei edições antigas/coleções públicas.

## Observações de uso público

- A autoria antiga em si está em domínio público.
- Traduções e edições recentes podem não estar.
- Project Gutenberg sinaliza status principalmente para EUA; para outros países, confirmar legislação local.

## Como usar este arquivo

1. Use os links de "ponto de entrada" (Wikisource autor / índice Gutenberg).
2. Priorize versões em TXT/HTML para reutilização e estudo.
3. Para máxima fidelidade filológica, use também os catálogos Perseus/SVF para fragmentos gregos/latinos.
