# Aethor DS Lab (localhost)

Playground web interativo para visualizar e testar componentes do design system fora do app Flutter.

## Como executar

Na raiz do projeto, rode:

```bash
python3 -m http.server 8080
```

Depois abra:

- http://localhost:8080/design-system-lab/

## Estrutura

- `index.html`: layout e componentes simulados.
- `styles.css`: tokens visuais e estilos.
- `app.js`: interações (favorito, estados, CTA loading, toasts).
- `docs/`: documentação migrada de branding/design system.
