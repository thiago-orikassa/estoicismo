# Entrega BE-01 + BE-06 - 2026-02-14

## Escopo
- BE-01: provisionamento backend dev/stage com migrações iniciais e runtime padronizado.
- BE-06: logs estruturados e métricas operacionais disponíveis.

## Entregas técnicas

### BE-01
- Script de provisionamento por ambiente:
  - `backend/scripts/provision.mjs`
- Script de execução por perfil:
  - `backend/scripts/start-profile.mjs`
- Novos scripts npm:
  - `npm run provision:dev`
  - `npm run provision:stage`
  - `npm run start:dev`
  - `npm run start:stage`
- Resultado do provisionamento:
  - aplica migrações em `schema_migrations`
  - garante seed e arquivos de dados mínimos
  - gera relatório em `.runtime/<profile>/provision-report.json`

### BE-06
- Logger estruturado JSON:
  - `backend/src/logger.mjs`
- Instrumentação de request em `backend/src/server.mjs`:
  - evento `http_request_completed`
  - campos: `request_id`, método, path, status, duração, ip, user_id
- Endpoint de métricas operacionais:
  - `GET /v1/observability/metrics`
  - proteção opcional via `STOIC_OBSERVABILITY_TOKEN`

## Critérios de aceite atendidos
- BE-01: ambientes dev/stage provisionáveis com migrações aplicadas.
- BE-06: logs estruturados emitidos e métricas consultáveis por endpoint.
