# Metas e Baseline v1 (PS-02)

Data: 2026-02-14

## Métricas Norte
- D1 Retention = usuários ativos no dia seguinte / novos usuários no dia 0
- D7 Retention = usuários ativos no sétimo dia / novos usuários no dia 0
- Open Rate Push = aberturas originadas de push / pushes entregues
- Daily Check-in Rate = check-ins completos / DAU
- First-Week Value = usuários com >=3 favoritos em 7 dias / novos usuários

## Metas iniciais (beta)
- D1 >= 35%
- D7 >= 15%
- Open Rate Push >= 25%
- Daily Check-in Rate >= 30%
- First-Week Value >= 20%

## Baseline operacional
Sem dados históricos. Baseline inicial será fechado após 100 usuários no beta.

## Governança
- Janela de leitura: coorte semanal
- Fonte de verdade: analytics events versionados
- Revisão: semanal no ritual de Review
